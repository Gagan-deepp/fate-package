# 🌿 Fate (命运)

> **"A warm breeze for your Express journey."**

`fate-cli-core` is a simple, aesthetic wrapper for Express.js, designed to bring a sense of calm and order to your backend development. Just like a sunny day, it provides a bright and organized starting point for your next project.

Developed with a Korean sense of **"Jeong" (정)** — a warmth that comes from thoughtfulness and care for the developer's experience.

---

## ✨ The Inevitable Features

- **Fate's Design**: A meticulously crafted directory structure that feels like it was meant to be.
- **Warmth in Logic**: Pre-configured middleware (JWT, Error Handling, Logger) that cares for your security and stability from Day 1.
- **Lightweight Core**: Pure Express power, wrapped in aesthetic simplicity.


## 🚀 Quick Start

Initialize your project with a single command:

```bash
npx fate-cli-core
```

Then, follow the warm path ahead:

```bash
npm install
npm run dev
```

## 📂 The Garden (Project Structure)

When you run Fate, it creates a cozy space for your code:

```text
src/
├── server.ts         # The heart of your application
├── controller/       # Thoughtful logic handlers
├── middleware/       # Guardians of your routes (JWT, Error, Logger)
├── routes/           # Clear pathways for your API
├── service/          # Dedicated workers for your data
└── ...
```

## 🍵 Philosophy

Development shouldn't be cold or mechanical. Fate is inspired by:
- **Sobak (소박함)**: The beauty of simplicity and lack of luxury.
- **Pyeon-an (편안함)**: Providing a comfortable environment where code can grow naturally.
